# 「石取りゲーム」のルール
N*Nの盤面でm個とったら勝ちのゲームを考えて
Nとmの規則性を見つけると

はじめは
- パス無し
- 着手禁止手(合法手がないならば負け)
- コウ,超コウ禁止
- 1つでもとったら勝ち(m=1)


を考慮する


はじめはm=1ではじめると解析が楽
